# ✨ Key Features
- **QRCode card**
- **Desktop and Mobile version**

# ⚠️ Prerequisites
- Node.js
- Node Package Manager
- React

# 📃 Setup
- Clone the repository using `git clone git@github.com:Tamunokuro/QR-Code-component.git`
- cd into the directory
- Run `npm install`
